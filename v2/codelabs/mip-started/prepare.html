如何开发一个 MIP 页面

如何开发一个 MIP 页面

本示例针对初次尝试开发 MIP 页面的开发者，将演示如何利用官方提供的一系列工具快速搭建一个 MIP 项目，开发一个 MIP 页面。

2. 准备环境

  1. 安装/升级 mip 命令行工具

    1
    npm install mip2 -g

    输入mip2 -V，若能正常显示版本号，说明已经安装成功。

    如果不成功可以在根目录尝试

    1
    sudo npm install mip2 -g

  2. 在合适的目录新建项目并命名，例如本教程中的 myproject

    1
    mip2 init

  3. 进入目录并启动开发服务器

    1
    mip2 dev

  4. 访问 http://127.0.0.1:8111/example/index.html 将看到初始 example 示例页面

下一节我们会学习初始化的项目的文件结构及 MIP 页面的内容结构及开发方式。
