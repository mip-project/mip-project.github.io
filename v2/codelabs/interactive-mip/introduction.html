如何打造可交互的 MIP 站点

如何打造可交互的 MIP 站点

本示例针对在 HTML 页面有通过数据驱动更新 DOM 节点需求的开发者

概述

前端中 数据驱动 的概念相信大家并不陌生：即不直接操作DOM节点，通过事先配置数据，并将 DOM 节点与数据绑定，通过触发数据的更新来触发 DOM 节点的更新。

MIP 提供的 mip-bind 的机制旨在帮助开发者打造通过数据驱动的、可交互的 MIP 站点。

通过本教程您可以学到

如何使用 mip-data 标签来初始化数据 如何使用绑定数据的指令 m-bind/m-text 如何使用 MIP.setData 方法来修改数据
